Please read the steps to execute the scripts:
==============================================
1) Import the project into eclips as Maven Project.
2) Go to pageObjects.properties file into /src/test/resources/ package
3) Change the :-
   BrowserType = chrome / firefox / ie
   browserDriver= webdriver.chrome.driver / webdriver.gecko.driver / webdriver.ie.driver
   
4) Execute your scripts by running the java class from /src/test/java/testScripts package.
